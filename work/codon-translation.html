Codon Translation

Educational Web App


Wanting to hone my skills in Javascript and UI design, I created an educational lookup tool that shows you the corresponding amino acid when you enter an mRNA codon sequence. The tool shows basic information about the amino acid such as the skeletal structure, other codons that code for the same amino acid, plus the commonly-used 1 and 3 letter abbreviations.

Please try it out for yourself here!
